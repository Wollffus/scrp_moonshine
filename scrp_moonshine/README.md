# scrp_moonshine
Moonshine Job made for REDEM:RP (redem/redem_roleplay)

# Installation
1. Put scrp_moonshine to your resource folder.
2. Add "scrp_moonshine" in your "server.cfg".
3. In order to start the moonshine job the player needs to have the job set to moonshiner / admin command example: /setjob <player_id> moonshiner 0
